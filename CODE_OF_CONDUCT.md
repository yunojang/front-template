# Code of Conduct

이 프로젝트의 참여자는 모두 아래 원칙을 따릅니다.

## 1. 존중

- 모든 팀원은 서로를 존중하며, 질문과 피드백을 환영합니다.
- 지식 수준이 다르더라도 비하하는 언어를 사용하지 않습니다.

## 2. 커뮤니케이션

- 의사소통은 이해하기 쉬운 한글을 기본으로 하며, 영문 기술 용어는 괄호로 병기합니다.
- 오해 소지가 있는 경우 문맥을 충분히 제공하고, 회의록/요약을 남깁니다.

## 3. 협업

- 과도한 야근이나 무리한 일정 요구를 지양합니다.
- 리뷰는 코드 품질 향상에 초점을 맞추고, 인신공격을 하지 않습니다.
- 의견이 갈릴 때는 자료 조사와 ADR 기록을 통해 합의합니다.

## 4. 책임

- 각자의 변경 사항에 책임을 가지며, 문제가 발생하면 해결을 돕습니다.
- 개인적인 사유로 작업이 어려울 경우 가능한 빨리 팀에 공유합니다.

## 5. 포용

- 성별, 나이, 국적, 경험 수준과 관계없이 모두에게 열린 공간을 지향합니다.
- 부적절한 행동이나 언어를 목격하면 팀 리드와 즉시 공유합니다.

모든 참가자는 이 규칙을 준수할 책임이 있으며, 위반 시에는 프로젝트 참여 제한 등 적절한 조치를 취할 수 있습니다.
